Front-End Web Technologies

A web browser displays a web page using

  • HTML
  • CSS
  • JavaScript (ECMAScript)

HTML

The Hypertext Markup Language uses tags to represent parts of the content

For example 'p' is a paragraph, 'ul' is an unordered list (of list items) etc.

Most tags need an open and a close e.g. <span>...</span>

A few tags are 'self-close' tags e.g. <br/>

Tags may have attributes

CSS

Cascading Style Sheets bring style to the content

  1. ID styles apply manually to one tag
  2. Class styles apply manually to many tags
  3. Tag styles apply automatically to matching tag
